# StarShield Affiliate Portal – Project Package

This package contains the full specification and assets for building the **StarShield Affiliate Portal** with two roles only:
- **Affiliate (User)**
- **Admin**

## Files

- `SRS.md` – Full Product Requirements & System Specification (detailed).
- `Architecture.mmd` – Mermaid flowcharts and sequence diagrams.
- `ERD.mmd` – Mermaid ER diagram.
- `Database_Schema.sql` – PostgreSQL schema (core tables, indexes).
- `API_Spec.yaml` – OpenAPI 3.0 spec for the core HTTP APIs.
- `Commission_Rules_Examples.json` – Example commission rule configurations.
- `architecture_high_level.png` – High-level architecture diagram.
- `flow_click_order_commission.png` – Click → Order → Commission flow diagram.
- `.env.example` – Example environment variables.

> Mermaid files (`*.mmd`) can be pasted into any Mermaid-compatible renderer (e.g., editors or docs).
> PNG diagrams are included for quick sharing.
